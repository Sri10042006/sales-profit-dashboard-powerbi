## 📌 Project Overview
This project analyzes a Superstore retail dataset (999+ transactions) 
to uncover key business insights around sales performance, 
profitability, and customer behavior across India.

Built using Microsoft Excel (data source) and 
Power BI Desktop (data transformation + visualization).

## 🔍 Key Business Insights
- 📦 4K Total Orders generating ₹53M in Total Sales and ₹11M in Total Profit
- 🌍 East & West regions lead in sales (~₹13.6M each)
- 🗂️ Office Supplies is the top-selling category (₹18M)
- 👤 Consumer segment drives the highest share of sales (~34%)
- 💳 Regular customer type generates the highest profit
- 🔘 Dynamic slicers for Profit Level and Sales Level drill-down

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| Microsoft Excel | Raw data storage & source |
| Power BI Desktop | Data cleaning, modeling & visualization |
| Power Query Editor | Data transformation (19 columns, 999+ rows) |

## 📊 Dashboard Features
- ✅ KPI Cards — Total Orders, Sales, Profit, Quantity
- ✅ Region Wise Sales — Horizontal bar chart
- ✅ Category Analysis — Sales vs Profit comparison
- ✅ Time Series — Sales & Profit trend by Order Date
- ✅ Customer Segment — Donut chart breakdown
- ✅ Customer Type Profit — Treemap visualization
- ✅ Interactive Slicers — Filter by Profit & Sales Level
- ✅ Data Cleaning — Done via Power Query Editor
